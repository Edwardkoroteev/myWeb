XV6 Unix Operating System

What is it?: xv6 is a modern reimplementation of Sixth Edition Unix in ANSI C for multiprocessor x86 systems. It is used for pedagogical purposes in MIT's Operating Systems Engineering. (click here to find out more)

Changes I Made to XV6

  • System Calls
  • Date Command
  • Process List
  • UID, GID, PID
  • Process Execution Time
  • PS Command
  • Time Command
  • Multi-Level Feedback Scheduler
  • File Protection
